# Operating System

Use the latest LTS version of Ubuntu (currently 18.04.2): 

* https://www.ubuntu.com/download/desktop

Create a bootable USB device from the `iso` file and install it on the target machine:

* https://tutorials.ubuntu.com/tutorial/tutorial-create-a-usb-stick-on-ubuntu
* https://tutorials.ubuntu.com/tutorial/tutorial-create-a-usb-stick-on-windows
